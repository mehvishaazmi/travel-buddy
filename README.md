
# Project Title

# TravelBuddy – AI-Powered Trip Planner & Expense Manager 🚀


## Description
TravelBuddy is a full-stack web application that helps users plan trips, manage group expenses, and collaborate in real-time.

It includes AI-powered trip planning, expense splitting, payment integration, and live chat functionality — all in one platform.

This project demonstrates real-world full-stack development using modern technologies.
## Features

✨ Features

- 🔐 Authentication (Clerk – Google + Email login)
- 🧳 Create & manage trips
- 👥 Add and manage trip members
- 💸 Track and split expenses automatically
- 💳 Razorpay payment integration
- 🤖 AI Trip Planner (generate travel plans)
- 💬 Chat system for group communication
- 📊 Expense summary & balances
- 🌐 Fully responsive UI


## Tech Stack

Frontend:
- Next.js (App Router)
- Tailwind CSS
- Shadcn UI

Backend:
- Next.js API Routes
- Supabase (Database)

Authentication:
- Clerk

Payments:
- Razorpay

AI:
- OpenAI / OpenRouter API

## Usage/Examples

- Sign up / Login using Clerk
- Create a new trip
- Add members to your trip
- Add expenses and track spending
- Split expenses automatically
- Make payments via Razorpay
- Use AI Trip Planner for suggestions
- Chat with group members
